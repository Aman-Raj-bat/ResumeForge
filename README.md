# ResumeForge

A full-stack web application to build professional resumes.

## Tech Stack
- **Frontend**: React (Vite), React Router DOM, Tailwind CSS, Axios, Framer Motion, React Hook Form, Zustand, Lucide React
- **Backend**: Node.js, Express.js, MongoDB (Mongoose)

## Installation
1. Clone the repository
2. Install client dependencies: `cd client && npm install`
3. Install server dependencies: `cd server && npm install`
4. Create `.env` files for both client and server based on the provided variables.

## Run Commands
- **Frontend**: `cd client && npm run dev`
- **Backend**: `cd server && npm run dev`

## Folder Structure
- `client/`: Contains the React frontend application.
- `server/`: Contains the Express backend API and server logic.
